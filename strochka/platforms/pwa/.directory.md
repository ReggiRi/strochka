# PWA Platform

Прогрессивное веб-приложение. Основная платформа для разработки и тестирования.

## Ключевые файлы

- `index.html` — точка входа с CSP-заголовками
- `src/main.ts` — инициализация ядра и модулей
- `vite.config.ts` — сборка Vite
- `package.json` — зависимости (codemirror, marked, dompurify)

## Связи

- Input: модули из `../../modules/editor/`, ядро из `../../kernels/typescript/`
- Output: собранное PWA в `dist/`
