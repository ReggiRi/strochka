# Editor Module

Модуль редактора и превью Markdown. Реализует интерфейс Module из KERNEL_SPEC.md.

## Ключевые файлы

- `editor.ts` — обёртка CodeMirror 6 с интеграцией через события ядра
- `preview.ts` — рендер Markdown в HTML (marked + DOMPurify)
- `styles.css` — сетка layout и базовые стили редактора

## Связи

- Input: Kernel (подписка на editor:mode)
- Output: editor:changed (при каждом изменении текста)
- Зависит от: `kernels/typescript/` (типы Kernel, Module)
