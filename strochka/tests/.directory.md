# Тесты STROČKA

Содержит кроссплатформенные конформные тесты для проверки соответствия ядер спецификации.

## Ключевые файлы

- `conformance/test-suite.json` — 13 сценариев в JSON-формате (агностик относительно языка)
- `conformance/runner.ts` — раннер на TypeScript, загружает сценарии и выполняет на KernelImpl

## Связи

- Input: ядро из `kernels/typescript/`
- Output: отчёт о прохождении тестов (stdout), код возврата 0/1
