# Конформные тесты

Кроссплатформенный тест-сьют для проверки соответствия ядра спецификации KERNEL_SPEC.md.
Формат JSON позволяет запускать сценарии на любом языке.

## Ключевые файлы

- `test-suite.json` — 13 сценариев: регистрация, эмит, жизненный цикл, обработка ошибок
- `runner.ts` — раннер на TypeScript, читает JSON и выполняет на KernelImpl

## Связи

- Input: `test-suite.json` (сценарии), `../../kernels/typescript/src/kernel.ts` (ядро)
- Output: PASS/FAIL для каждого сценария, exit code
