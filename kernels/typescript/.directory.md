# TypeScript Kernel (Reference Implementation)

Reference implementation ядра STROČKA на TypeScript.
Служит эталоном для проверки контракта и тестирования модулей.

## Ключевые файлы

- `src/kernel.ts` — реализация Kernel и Module интерфейсов (~150 строк)
- `tests/kernel.test.ts` — 19 тестов: регистрация, эмит, жизненный цикл, ошибки
- `package.json` — зависимости (typescript, vitest) и скрипты

## Связи

- Input: контракт из `KERNEL_SPEC.md` (корень проекта)
- Output: типы Kernel, Module, Handler, Unsubscribe — используемые модулями и тестами
