# Git Module

Модуль контроля версий на isomorphic-git с lightning-fs.

## Ключевые файлы

- `git-interface.ts` — контракт Git (CommitEntry, Git)
- `git.ts` — реализация на isomorphic-git + lightning-fs
- `__tests__/git.test.ts` — тесты

## Связи

- Input: Storage (синхронизация документов в ФС перед коммитом)
- Output: git:commit — после успешного коммита
- Зависит от: storage
