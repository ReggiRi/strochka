# Sidebar Module

Модуль боковой панели с файловым деревом и управлением проектами.

## Ключевые файлы

- `sidebar.ts` — модуль с UI проектов и документов
- `sidebar.css` — стили файлового дерева

## Связи

- Input: Storage (через kernel.getModule), события project:*, document:*
- Output: editor:load — при клике на документ
- Зависит от: storage
