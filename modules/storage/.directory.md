# Storage Module

Модуль хранения проектов и документов. Реализует интерфейс Storage из KERNEL_SPEC.md.

## Ключевые файлы

- `storage-interface.ts` — контракт хранилища (Project, StoredDocument, Storage)
- `storage-indexeddb.ts` — реализация на IndexedDB
- `__tests__/storage.test.ts` — тесты

## Связи

- Input: Kernel (события)
- Output: events: project:created, project:deleted, document:changed
- Зависит от: нет (чистая абстракция)
