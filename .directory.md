# Корень проекта STROČKA

Монорепозиторий модульного редактора для писателей.
Содержит спецификацию ядра, reference implementation на TypeScript и платформенные сборки.

## Ключевые файлы

- `KERNEL_SPEC.md` — конституция: контракт ядра, контракт модуля, спецификация событий
- `README.md` — описание проекта и быстрый старт

## Поддиректории

- `kernels/typescript/` — reference implementation ядра на TypeScript
- `modules/` — переиспользуемые модули (будущее: editor, storage, git, export)
- `tests/conformance/` — кроссплатформенный тест-сьют для проверки ядер
- `docs/` — документация архитектуры и контрибьюции
- `platforms/` — платформенные сборки (будущее: PWA, JavaFX, Tauri)

## Связи

- Input: нет (корень проекта)
- Output: спецификация поступает в `kernels/`, модули используют контракт из `KERNEL_SPEC.md`
